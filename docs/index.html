🔍 MCP Debug Proxy

Intercept, log, analyze, and replay MCP server traffic.

Quick Install

pip install mcp-debug-proxy

# Record a session
mcp-debug --out session.json -- python my_server.py

# Generate a report
mcp-debug --report session.json

Features

  • Proxy Mode — sits between client and server, records all JSON-RPC traffic
  • Latency Analysis — per-method response times
  • Session Recording — full message log with microsecond timing
  • Export — JSON, text reports, Mermaid diagrams (paid)
  • Replay — replay sessions for regression testing (paid)
